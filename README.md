# BYRDOCS TUI

> BYRDOCS 集成命令行工具

- 以命令行应用实现 BYRDOCS 的文件整理。
- 具备简洁易用的 Terminal UI 界面。
- 兼容 Windows, Linux 和 MacOS。
- 以 .yml 文件保存文档元信息，并存储于 [archive](https://github.com/byrdocs/byrdocs-archive) 当中。
- 支持以下功能：
	- 初始化工作目录
	- 文件上传/下载
	- 文件元信息查询
	- 元信息编辑与更新
	- Git 操作
	- 文件比较与去重
	- ……
